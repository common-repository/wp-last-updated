WP lastupdated lets you easily add a shortcode or widget to you site which tells your visitors when the last time your site was updated.

== Description ==

The shortcode to display when your site was last updated is [WPlastupdated text = ""] where you can enter any text in the "". Built by [Epic Plugins](http://epicplugins.com)

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. The shortcode to use is [WPlastupdated text = ""]

== Frequently asked questions ==

= What is the shortcode =

You can show the last time your site was updated by using the shortcode [WPlastupdated] this will show the date and time your site was last updated. To show some text use the shortcode [WPlastupdated text = "This site was last updated: "]
